# performance-testing

🛠️ Configuration Steps

1. Update CSV Data Path
The test script relies on login.csv for user credentials. You must update the file path to match your local directory before running the test:

    - Open the .jmx file in Apache JMeter.

    -  In the left-hand panel (Test Plan tree), locate the CSV Data Set Config element.

    -  In the Filename field, click Browse and select the login.csv file located in this project folder.
